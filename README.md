# Translator App

## Setup

### Clone Repository

```
git clone https://github.com/aqib-git/translator.git
cd translator
```

### Install dependencies

```
composer install
```

### .env file
```
GOOGLE_API_KEY= // set google cloud translation api key here
```

### Start Application

```
php artisan serve
```

### Run tests
````
vendor/bin/phpunit
````
